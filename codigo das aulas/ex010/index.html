Um pouco sobre links

Para entender sobre links precisamos entender também sobre como funciona a busca realizada pelo google por exemplo.
Quando voce pesquisa algo no google ele vai percorrer todas as paginas que estão no seu indice, vai procurar pela palavra chave que você pesquisou em titulos ou na URL, vai procurar por sinonimos da palavra no seu site, quantas vezes a palavra chave aparece. Aqui entra a importancia de saber usar os títulos e a hora certa e onde colocar eles para que tudo isso favoreça o seu site.


Usando links externos

O que é um link externo??
Um link que vai apontar para um site externo, um site diferente do que o que está sendo usado.

Você pode acessar o meu repositório público no GitHub

Para realizar o processo demonstrado acima basta seguir esse passo a passo;

  1. Aqui voce vai colocar o texto que deseja ficar antes do link, ex:Você pode acessar o meu
  2. Logo depois irá adcionar a seguinte tag <a>
  3. automaticamente a tag <a> vai ficar dessa forma <a href="link externo de sua esscolha">
  4. Resto do texto caso queira algo mais.
  5. Fechar a tag </a>

  • A tag <a> e </a> serve como âncora e automaticamente vai inserir o href que vai ser aonde você vai colocar a URL do site desejado entre as aspas duplas.
  • Ao clicar no hiperlink o conteúdo que estava sendo exibido deixou de sesr exibido e uma nova pagina foi aberta no lugar na antiga, fazendo com que o usuário saia do nosso sites e vá para outro. Para controlar onde o site de destino, ou seja o hiperlink, vai abrir podemos o usar o tributo target que suporta os seguintes valores;

    • _blank: Vai abrir o link em uma nova janela em branco.
    • _self: Vai abrir o link na janela ou frame atual (padrão).
