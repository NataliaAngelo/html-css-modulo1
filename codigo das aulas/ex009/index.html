Lista de Definições

Serve para colocar o termo e suas definições, verifique os exemplos abaixo;

HTML
Linguagem de marcação para criação do conteúdo de um site
CSS
Lingugem de marcação para criação do design de um site.
JavaScript
Linguagem de programação para criação de interatividade de um site.
Essa lista pode ser criada da seguinte maneira; 
    <dl> (dl significa Lista de definição)

    <dt> </dt>(dt significa termo)
    <dd> </dd>(dd significa descrição)

    </dl>

Mecanismos de busca, o google por exemplo, na hora de realizar uma busca por um determinado termo, sites que usaram o <dt> e o <dd> são favorecidos em relação aos sites que não usaram as tags.
